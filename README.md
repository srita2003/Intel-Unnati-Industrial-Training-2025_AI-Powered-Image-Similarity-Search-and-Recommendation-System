# Intel-Unnati-Industrial-Training-2025_AI-Powered-Image-Similarity-Search-and-Recommendation-System
AI-powered image similarity search for fashion. Uses a Triplet Network to learn visual embeddings, enabling real-time, content-based recommendations across large datasets. Supports category-wise search (shoes, apparel, accessories) for e-commerce and digital catalogs. Python, PyTorch, CNNs.
